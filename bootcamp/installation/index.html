01. Installation

Our first step is to create the web app and setup our local environment. This Bootcamp assumes you have PHP installed on your system.

Installing Laravel

We'll create a fresh Laravel project using Laravel's installer. If you don't yet have the installer, you may install it using Composer:

composer global require laravel/installer

Next, we'll create the fresh Laravel project using the Hotwire Starter Kit:

laravel new turbo-chirper --using=hotwired-laravel/hotwire-starter-kit --pest

Answer "no" to the question about NPM. After the project has been created, start Laravel's local development server using the Composer dev script:

cd turbo-chirper/
composer run dev

Once you have started the Artisan development server, your application will be accessible in your web browser at http://localhost:8000.
